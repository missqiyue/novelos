# 战斗与情感写作模板（通用｜细腻+高情绪｜含亲密戏Lv1-5）

> 用途：把“战斗细腻、情感上头、亲密有尺度且不水”的写法固化成可复制模板。  
> 原则：**动作要可想象，情绪要可触摸，关系要有阶段；亲密写到门槛，黑屏用后果交付。**

---

## 1) 战斗写法（从“像打游戏”改成“像在现场”）

### 1.1 五层镜头（从外到内）
1) **空间层**：地形、掩体、距离、光线（读者先知道“人在哪”）  
2) **节奏层**：快慢、停顿、换气（让读者“喘得上气”）  
3) **动作层**：冲刺/借力/格挡/反打/撤离（动作链要完整）  
4) **代价层**：疼、麻、喘、视线发黑、耳鸣（每场硬仗至少1项代价）  
5) **意志层**：选择（救谁/追谁/弃谁），把情绪塞进选择里

### 1.2 一场战斗的最小结构（8步）
1) 目标：要抢什么/救谁/拖多久  
2) 对手优势：对方凭什么赢（武器/人数/地形/规则）  
3) 试探：第一下交手，暴露差距  
4) 破局点：主角发现漏洞（不靠嘴炮）  
5) 反制：动作链写清楚（别一句“他爆发了”）  
6) 代价：受伤/丢资源/错过窗口  
7) 结果：赢但不圆满（为下一章埋钩子）  
8) 钩子：更强敌人/新规则/队友状态恶化

### 1.3 句子层技巧（去AI味）
- 少用“爆发/恐怖如斯/快到看不清”，多用具象：  
  - “枪机回弹顶在掌心，骨头像被锤了一下。”  
  - “他踩到碎屑，脚下一滑，肩膀旧伤像被撕开。”  
- 每场战斗至少一个“非战斗细节”：血混雨、手套粘汗、金属发烫、口腔铁锈味。

## 2) 情感写法（让读者追关系）

### 2.1 情感不是“甜”，是“代价与选择”
你要写的是：  
- **我想要你**（欲望）  
- **我怕失去你**（恐惧）  
- **我不得不伤到你**（大局与代价）  
三者循环，读者才会追。

### 2.2 情感场景三件套（每30章至少一次）
1) **并肩**：一起扛一件事（战场/听证/撤离/办案）  
2) **裂缝**：价值观冲突（救一城还是救一人/规矩还是情义）  
3) **修复**：不是道歉，是“共同承担后果”

### 2.3 细腻表达模板（动作替代宣言）
- 她说“没事”，手指却在杯沿扣出一圈白。  
- 他想伸手扶她，停在半寸处，转而把药放到她掌心。  
- 争吵结束不是“我爱你”，而是：他把自己的证件/底牌交给她保管。

---

## 3) 事业线与感情线“交叉爆点”（强制用）

### 模式A：事业赢→感情输
- 主角赢下一局，但必须牺牲一个承诺（爽里带刀）。

### 模式B：感情赢→事业受损
- 主角救她/保她，结果丢掉资源点或被问责（甜里带血）。

### 模式C：共同背锅→共同翻盘
- 两人被栽赃；他们用硬点/真相把桌子掀回来（最容易上头）。

---

## 4) 章末钩子（情感线也要钩）
- 她没回消息，但发来一张旧照片。  
- 他明明活着回来了，却把门锁上不让她进。  
- 她第一次叫他名字，不叫职务。

## 5) 亲密戏（同女频规格｜高张力但不直白｜不只主CP）

> 写法总纲：**细节写到“情绪阈值”，动作在门槛处停下，用暗示与后果完成闭环。**

### 5.1 三条底线（建议写死）
1) 只写**成年人**、**自愿**、**可撤回**的亲密。  
2) 不描摹明确器官与具体性行为过程（用“暗场/省略/黑屏”）。  
3) 亲密戏必须服务剧情：推进关系阶段、制造裂缝或修复，不做无意义灌水。

### 5.2 亲密戏强度等级（1-5）
> 最高级别做到**强暗示+黑屏+后果**，仍不写直白色情过程与器官细描。
- **Lv1 触碰**：指尖/衣角/发簪/手背擦过，心跳与克制为主。  
- **Lv2 试探吻**：浅吻/咬唇边缘/呼吸贴近，及时刹车。  
- **Lv3 失控吻**：更长的吻、压迫式贴近、克制与放纵拉扯；止于门槛。  
- **Lv4 黑屏前夜**：衣襟/束带/发簪等“乱”，灯影/帷帐/门闩/车窗雾等氛围拉满；切黑屏，用次日细节交代。  
- **Lv5 最高级（强暗示+黑屏+后果）**：  
  - 写到门槛：呼吸失控、指腹探索、衣料滑落一寸、身体反应（只写感受不写部位）；  
  - 立刻黑屏（转场到外部危机/倒计时/有人敲门）；  
  - 用后果交付：口脂痕/领口扣错/手腕红印/披风或外套落地/他嗓音哑/她走路慢半拍/称呼改变；  
  - 必须绑定剧情：导致【盟约更深/背刺更痛/错过更狠/情报泄露】其一。

### 5.3 亲密戏三段式结构（可套用）
1) **引火（拉近）**：称呼变化、站位靠近、温度与呼吸、指尖触碰（先情绪）。  
2) **越界（失控一瞬）**：一个吻/一个拥抱/扣住手腕（写“犹豫→放纵”）。  
3) **落锤（黑屏/后果）**：写到门槛处停下→切转场→用次日/外部视角把后果落地。

### 5.4 “不直白但很上头”的细节库
- 触感：衣料摩擦、手心的汗、指腹擦过唇角、颈侧一瞬战栗  
- 听觉：呼吸乱、玉佩轻撞/拉链轻响、烛芯噼啪/远处警报  
- 视觉：灯影摇晃、发簪斜了、领口松开一指、指尖沾到一抹口脂  
- 禁忌感：门外脚步声、隔墙有人、倒计时在跳、他压低嗓音叫她名字

### 5.5 成人向“镜头感”写法（像成人小说/成人向镜头，但不写私密器官）
> 目标：写出更强的性张力与挑逗感（镜头语言、动作链、停顿、呼吸、推拉），但**不出现私密器官**、不写具体性行为过程。

**激进上限（可以更露骨，但仍不越线）**  
- ✅ 可写：脱衣/解扣/衣物滑落、半裸、背/肩/锁骨/腰线/腿部线条、被压在门板/桌沿/车座的压迫感、湿热呼吸与吞咽、指节与掌心的力度、口脂/汗/香味、求饶与命令（但必须自愿且可撤回）。  
- ❌ 不写：私密器官本体；也**不使用任何“替代词/黑话/隐语”**去指代私密器官；不写具体性行为过程与动作步骤（包括“进入/抽动/高潮”等过程描写）。  

**（1）动作链：眼神→距离→触碰→停顿→黑屏→后果**  
- 先让角色“看见彼此”（对视/回避/盯住），再推进距离（一步、一寸、一呼吸）。  
- 触碰只写“感受与反应”，不写器官细节。  
- 到门槛立刻切黑屏（外部危机/倒计时/门外脚步/灯灭），再用后果交付。

**（2）推拉：让“允许/拒绝/再允许”可见**  
- 必须出现“许可台词”或“可撤回动作”：  
  - “可以吗？”/“说停我就停。”/“你现在还能反悔。”  
  - 对方必须明确回应（点头/拉近/说“别停”/反手扣住）。

**（3）氛围替代直白**  
- 用：衣料、扣子、指节、体温、气味、灯影、门锁声、呼吸乱掉、嗓音变哑、脚步踉跄半拍。  
- 不用：器官名称、体位与过程说明。

**（4）黑屏后的“后果落锤”清单（任选1-3条）**
- 领口扣错/发簪不见/披风或外套落地  
- 手腕红印/指节擦伤/唇上破皮（可混入战斗伤更自然）  
- 嗓音哑/对视回避/称呼升级或退回敬称  
- 关系结算：当章“站队/盟约/误会/背锅/交易条件”落地

**（5）可直接套用的句式库（不露骨）**
- “他停了一瞬，像在等她最后一次反悔。”  
- “她想推开，手却先抓住了他的袖口。”  
- “门锁轻响，空气忽然变得太近。”  
- “灯影晃了一下，她的呼吸先乱。”  
- “次日他替她把扣子扣好，像把一场失控写进了规矩里。”

### 5.6 “要多一些”怎么写（不只主CP｜多对CP/多组合）
> 方法：**主CP负责关键节点亲密**；**副CP负责高频日常亲密**；**灰度/反派负责危险暧昧张力**。

#### 5.6.1 密度建议（按卷50章）
- **主线亲密（主CP相关）**：每卷 **6—10场**（Lv4-5建议2—4场，其余Lv2-3铺情绪）。  
- **副CP亲密**：每卷 **4—8场**（以Lv2-4为主，负责“喘息与上头”，同时带出情报/误会/站队）。  
- **危险暧昧**：每卷 **1—2场**（只写张力，不让关系失控到毁人设）。

#### 5.6.2 写多不水：每场只做一件事
每场亲密戏只推动一个核心结果（选一）：
1) 关系阶段上升（称呼变化/默认关系/交底牌）  
2) 关系阶段下降（误会成立/切割/背刺）  
3) 主线推进（拿情报/换筹码/逼站队）  
4) 角色代价落地（伤、名声、反噬、门规、军令、问责）

## 6)（可选）男频“多女主/女副角上头戏”模块（暗恋/明恋/勾引/表白/牺牲救援）

> 适用：末世/副本/玄幻/都市强者等男频常见结构。  
> 目标：满足“被认可/被偏爱/强者围绕”的爽点，但不写成油腻后宫、也不抢主线推进。

### 6.1 五个功能位（先定槽位，再写戏）
1) 正宫/共署名位（稳定托底）  
2) 女强战友位（专业能力救命/托底）  
3) 危险诱惑位（交易式暧昧/试探底牌）  
4) 对手欣赏位（强者认可/合作背刺边缘）  
5) 牺牲救援位（一次高情绪结算，但必须有回响）

### 6.2 三条硬规则（写多不乱）
1) 每3章最多推进1条情感线（否则主线被稀释）。  
2) 每场情感戏必须带“剧情用途”（拿情报/逼站队/落代价/修复裂缝/反派变招）。  
3) 牺牲不是工具人：牺牲前至少铺3次“她自己的目标与选择”；牺牲后必须产生硬点回响（证据/授权/战绩/制度改变其一）。

## 7) 亲密戏强制绑定规则（新增｜必须执行）

### 7.1 绑定三问（每场亲密戏写之前必须回答）

每写一场亲密戏，必须回答这3个问题：

| 问题 | 答案格式 |
|------|----------|
| **1) 这场亲密推进了哪条故事链？** | 主线目标链 / 敌人链 / 硬点链 / 情感链 / 伏笔链（选1） |
| **2) 这场亲密的代价是什么？** | 当章必须落地（伤/名声/信任/承诺/把柄/情报泄露） |
| **3) 这场亲密的后果如何被第三方利用？** | 制造新冲突（误会/抓现行/站队/背刺/要挟） |

**如果3个问题有任何1个答不出来，这场亲密戏暂缓，先想清楚。**

### 7.2 亲密戏与故事链绑定表

| 故事链类型 | 亲密戏能做什么 | 代价必须是什么 |
|------------|----------------|----------------|
| **主线目标链** | 推进目标进度（拿到证据/通过审核/获得授权） | 暴露弱点/耽误时间/错过窗口 |
| **敌人链** | 获取情报/离间对手/制造误会 | 被反试探/暴露身份/信任裂痕 |
| **硬点链** | 落袋筹码/确认盟约/交换条件 | 承诺绑定/失去退路/名声代价 |
| **情感链** | 关系升级（称呼/默认/交底牌） | 裂缝加深/选择困境/被迫站队 |
| **伏笔链** | 回收伏笔/验证身份/触发条件 | 伏笔暴露/被第三人看见/代价显现 |

### 7.3 亲密戏落点检查清单（每场写完打勾）

- [ ] 回答了"绑定三问"（3个问题都有答案）
- [ ] 代价当章落地（不是"以后再说"）
- [ ] 有第三人知道的风险（误会/抓现行/要挟）
- [ ] 爽点与主线绑定（不是"为了甜而甜"）
- [ ] 章末钩子与亲密后果关联

### 7.4 亲密戏时间线绑定法（防止密集恐惧）

> 适用于：有明确时间线约束的题材（副本流/科举流/朝堂审案等）。

```markdown
## 时间线绑定规则

### 副本流/灾变流（时间压缩型）
- **非窗口期**（副本外）：日常训练/政治博弈/资源筹备 → 亲密戏主战场（Lv3-5密集）
- **窗口期前夜**：高压下的情感爆发 → Lv4-5高密度区
- **副本进行中**：生死边缘的短暂触碰 → Lv1-2点缀
- **副本结算后**：庆功/复盘/代价清算 → Lv3-4（甜后接刀）

### 朝堂流/权谋流（事件驱动型）
- **日常铺垫期**：收集证据/布局/练兵 → 亲密戏日常化（Lv1-3）
- **关键节点前夜**：听证/审案/大战前 → Lv4-5爆发
- **事件爆发中**：局势紧张 → Lv1-2点缀
- **事件结算后**：论功行赏/清算/站队 → Lv3-4（甜苦交织）

### 科举流/升级流（阶段驱动型）
- **日常修炼期**：练功/学习/社交 → 亲密戏日常（Lv1-3）
- **升级/考试前夜**：压力测试 → Lv4-5
- **升级/考试中**：限时紧张 → Lv1-2点缀
- **升级/考试后**：收获/代价清算 → Lv3-4
```

### 7.5 亲密戏"三章一批"轮转模板

> 解决"连续写亲密戏会腻"和"一直不写会掉粉"的问题。

```markdown
## 3章一批亲密戏分配（按批次循环）

### 批次A（情感批次）
- 第N章：亲密戏核心章（Lv4-5）
- 第N+1章：后果消化（第三人知道/误会/要挟）
- 第N+2章：关系修复 or 裂缝加深（必须选一）

### 批次B（事业批次）
- 第N章：亲密戏点缀（Lv1-2，蜻蜓点水）
- 第N+1章：事业线推进（亲密后果影响）
- 第N+2章：新冲突浮现（亲密戏伏笔回收）

### 批次C（过渡批次）
- 本批无亲密戏（或仅Lv1暗示）
- 用于：
  - 承接上批后果
  - 为下批情感批次做铺垫
  - 调节节奏（防止审美疲劳）

### 轮转顺序
A → B → C → A → B → C ...

每3批（9章）至少包含：
- 1次Lv4-5核心亲密
- 2次Lv1-2点缀
- 0-1次无亲密（过渡）
```

### 7.6 快速检查：这场亲密戏有没有"白写"？

```markdown
## 亲密戏白写检测（5秒检查）

如果这场亲密戏删掉后：
- 剧情仍然能推进 → ❌ 白写
- 人物关系没有变化 → ❌ 白写
- 没有产生任何代价/后果 → ❌ 白写
- 任何代价/后果都"以后再说" → ❌ 白写

如果这场亲密戏删掉后：
- 某条故事链无法推进 → ✅ 有效
- 人物关系阶段明确变化 → ✅ 有效
- 当章就有代价/后果落地 → ✅ 有效
- 第三方反应制造新冲突 → ✅ 有效
```

## 8) 情绪节拍关联（亲密戏在波浪中的位置）

### 8.1 亲密戏的最佳插入点

| 节拍位置 | 适合亲密戏类型 | 情绪效果 |
|----------|----------------|----------|
| **压抑→释放转折点** | Lv4-5核心亲密 | 情绪释放最大化，读者"终于等到" |
| **释放→新压抑起点** | Lv3-4后果亲密 | 甜后立刻接刀，读者"心被抓住" |
| **新压抑中段** | Lv1-2点缀 | 紧张中的温柔，读者"更担心失去" |
| **对抗阶段** | 不适合亲密 | 节奏会断，不建议 |

### 8.2 情绪波浪+亲密戏组合示例

```markdown
## 标准波浪型 + 亲密戏（第N批）

| 章节 | 节拍 | 亲密戏安排 |
|------|------|------------|
| 第N章 | 【压抑】 | Lv1-2点缀（想靠近但被打断） |
| 第N+1章 | 【对抗】 | 无亲密（节奏紧张） |
| 第N+2章 | 【释放+新压抑】 | Lv4-5核心亲密（情绪释放最大化）+ 后果落锤 |
```
