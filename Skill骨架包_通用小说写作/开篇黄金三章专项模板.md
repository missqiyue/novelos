# 开篇黄金三章专项模板（网文生死线）

> 用途：解决"开篇流失率高"的问题。  
> 原理：网文开篇（前3章）是决定生死的黄金三章，必须在最短时间内抓住读者。

---

## 0) 黄金三章核心原则（写死）

1. **第1章必须让读者"停下来"**：身份反差+即时危机+当场翻盘
2. **第2章必须让读者"看懂主线"**：展示能力+明确目标+建立关系
3. **第3章必须让读者"追下去"**：小高潮+代价显现+更大威胁

---

## 1) 第1章：生死线（必须命中5项）

### 1.1 必须命中项

| 序号 | 必须命中 | 具体要求 | 检查 |
|------|----------|----------|------|
| 1 | **身份反差** | 重生/穿越/被弃/被冤/替身/被退婚 | [ ] |
| 2 | **即时危机** | 当夜就要死/被送人/被灭口/被审判 | [ ] |
| 3 | **当场翻盘** | 先赢一局，让读者看到主角能力 | [ ] |
| 4 | **情绪刀子** | ≤2句，埋下后续虐点 | [ ] |
| 5 | **章末钩子** | 更大的威胁/更深的阴谋 | [ ] |

### 1.2 第1章结构模板

```markdown
## 第1章《标题》结构

### 开头（前200字）
- 场景锚：在哪（一句话）
- 身份锚：主角是谁，什么处境（一句话）
- 危机锚：即将发生什么（一句话）

### 中段（800-1200字）
- 危机降临：具体的威胁（人/规则/资源）
- 主角反应：不是"意识到"，而是"动作链"
- 翻盘动作：用能力/信息差/规则反用
- 翻盘结果：可验证的胜利（拿到/逼出/翻转）

### 结尾（300-500字）
- 情绪刀子：≤2句（她最信的人，恰好是推她下去的人）
- 章末钩子：更大的威胁（但真正的危险才刚开始）

### 字数检查
- 总字数：1500-2500
- 开头200字内是否交代：谁+在哪+要干什么
```

### 1.3 第1章检查清单

```markdown
## 第1章检查清单

### 必须命中
- [ ] 身份反差：________
- [ ] 即时危机：________
- [ ] 当场翻盘：________
- [ ] 情绪刀子：________
- [ ] 章末钩子：________

### 禁止出现
- [ ] 无大段背景介绍（≤100字）
- [ ] 无"他意识到/他明白/他知道"
- [ ] 无"多年以后"式的时间跳跃
- [ ] 无超过3个新名词

### 快懂检查
- [ ] 开头200字内交代：谁+在哪+要干什么
- [ ] 第500字内出现第一个"动作"
- [ ] 第1000字内出现第一个"翻盘动作"
- [ ] 结尾有"必须看第2章"的理由
```

## 2) 第2章：理解线（必须命中4项）

### 2.1 必须命中项

| 序号 | 必须命中 | 具体要求 | 检查 |
|------|----------|----------|------|
| 1 | **展示核心能力** | 金手指/重生信息/特殊技能 | [ ] |
| 2 | **建立第一个关系** | 盟友/对手/信息枢纽 | [ ] |
| 3 | **明确主线目标** | 一句话说清"要干什么" | [ ] |
| 4 | **章末钩子** | 第一个障碍/第一个代价 | [ ] |

### 2.2 第2章结构模板

```markdown
## 第2章《标题》结构

### 开头（前200字）
- 承接上章钩子
- 场景转换（如果需要）

### 中段（800-1200字）
- 能力展示：用具体场景展示核心能力
- 关系建立：第一个重要角色登场
- 目标明确：主角说出/表现出"要干什么"
- 第一个障碍：不是一帆风顺

### 结尾（300-500字）
- 代价显现：能力使用/翻盘的代价
- 章末钩子：新问题/新威胁/新选择

### 字数检查
- 总字数：1500-2500
```

### 2.3 第2章检查清单

```markdown
## 第2章检查清单

### 必须命中
- [ ] 核心能力展示：________
- [ ] 第一个关系：________（盟友/对手/信息枢纽）
- [ ] 主线目标：________（一句话）
- [ ] 章末钩子：________

### 禁止出现
- [ ] 无大段设定讲解（规则≤2行）
- [ ] 无"作者视角"的背景介绍
- [ ] 无超过5个新名词

### 快懂检查
- [ ] 读者能说出"主角要干什么"
- [ ] 读者能说出"主角有什么能力"
- [ ] 读者能说出"主角面临什么障碍"
```

## 3) 第3章：追读线（必须命中4项）

### 3.1 必须命中项

| 序号 | 必须命中 | 具体要求 | 检查 |
|------|----------|----------|------|
| 1 | **第一次小高潮** | 小胜/小翻盘/第一个硬点落袋 | [ ] |
| 2 | **代价显现** | 胜利不是免费的 | [ ] |
| 3 | **更大阴谋/对手露出** | 让读者知道"这只是开始" | [ ] |
| 4 | **追读理由确立** | 读者"必须追下去"的理由 | [ ] |

### 3.2 第3章结构模板

```markdown
## 第3章《标题》结构

### 开头（前200字）
- 承接上章钩子
- 进入"第一个小高潮"的铺垫

### 中段（800-1200字）
- 小高潮场景：具体的对抗/翻盘
- 爽点落袋：可验证的胜利（签字/按印/封存/编号）
- 代价显现：胜利的代价（资源/名声/关系）
- 更大威胁：对手变招/更高层敌人露出

### 结尾（300-500字）
- 追读钩子：让读者"必须追下去"
- 可以是：新问题/新代价/新威胁/新选择

### 字数检查
- 总字数：1500-2500
```

### 3.3 第3章检查清单

```markdown
## 第3章检查清单

### 必须命中
- [ ] 第一次小高潮：________
- [ ] 代价显现：________
- [ ] 更大威胁：________
- [ ] 追读理由：________

### 爽点检查
- [ ] 本章有≥1个"爽点"（拿到/逼出/翻转）
- [ ] 爽点有"可验证形式"（不是情绪胜利）
- [ ] 爽点有"代价"

### 追读检查
- [ ] 读者能说出"主角下一步要干什么"
- [ ] 读者能说出"主角面临什么更大威胁"
- [ ] 读者有"必须追下去"的理由
```

## 4) 黄金三章总检查

- [ ] 第1章结尾钩子在第2章开头承接

## 5) 黄金三章常见问题与解决方案

| 问题 | 症状 | 解决方案 |
|------|------|----------|
| 开篇流失率高 | 第1章追读率低 | 检查身份反差+即时危机+当场翻盘 |
| 第2章流失率高 | 读者看完第1章不看第2章 | 检查第1章结尾钩子+第2章能力展示 |
| 第3章流失率高 | 读者看完第2章不看第3章 | 检查第2章目标明确+第3章小高潮 |
| 整体流失率高 | 三章追读率持续下降 | 检查三章连贯性+爽点密度+代价显现 |

## 7) 黄金三章快速模板（可直接套用）

### 7.1 男频重生流

```markdown
## 第1章：重生+即时危机+当场翻盘
- 开头：重生醒来，发现距离灾变只有48小时
- 危机：上一世的内鬼已经在行动
- 翻盘：用重生信息提前布局，拿到第一份资源
- 钩子：发现内鬼比想象中更近

## 第2章：能力展示+建立关系+明确目标
- 能力：展示国库空间/规则解析能力
- 关系：联系特管局，建立第一个盟友
- 目标：在灾变前建立战略储备点
- 钩子：特管局内部有质疑声音

## 第3章：第一个副本+小高潮+代价
- 副本：第一个灾变副本开启
- 高潮：成功通关，拿到核心碎片
- 代价：消耗大量资源，被灰市盯上
- 钩子：内鬼的执行链第一次露出
```

### 7.2 女频重生复仇流

```markdown
## 第1章：重生+被冤+当场翻盘
- 开头：重生醒来，发现自己即将被定罪
- 危机：当夜就要被送人/被灭口
- 翻盘：抢到关键证据，暂时脱险
- 钩子：推她下去的人，是她最信任的人

## 第2章：能力展示+建立关系+明确目标
- 能力：展示命书线索/秘术能力
- 关系：与第一个男主（权/命/爱之一）相遇
- 目标：翻案复仇，夺回家族清白
- 钩子：发现翻案的关键证据在对手手中

## 第3章：第一个情感节点+小高潮+代价
- 节点：与男主的第一次"并肩/冲突"
- 高潮：拿到第一个关键证据
- 代价：付出名声/自由/信任的代价
- 钩子：发现更大的阴谋在后面
```
