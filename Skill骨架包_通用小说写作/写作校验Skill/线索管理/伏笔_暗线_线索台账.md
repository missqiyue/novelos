# 伏笔 / 暗线 / 线索台账（通用模板）

> 核心原则：**所有坑都要有ID**。  
> - “坑”不是一定要在本书填完；但必须在台账里标明：**回扣章**或**续作钩子去向**。  
> - 每3章：新增≤1条、回收≤1条（强制节制，防爆炸）。  

---

## 1) 台账字段定义（模板）

| 伏笔ID | 类型 | 首次出现章 | 表层呈现 | 真实含义/指向 | 触发条件（何时该回收） | 计划回扣（章/区间） | 证据载体（物/人/文书） | 牵涉人物/势力 | 状态 | 回扣去向 |
|---|---|---:|---|---|---|---|---|---|---|---|
| FB-___ | 坑/暗线/线索/钩子 |  |  |  |  |  |  |  | 未回收/回收中/已回收/改钩 | 本书/续作/外传 |

### 状态枚举（建议统一）
- **未回收**：已埋但尚未触发关键动作  
- **回收中**：已进入“可验证步骤”（按印/封存/受理编号/并线表）但因果未闭环  
- **已回收**：因果闭环或真相揭示，正文已落地  
- **改钩**：本书不回收，已明确写成“续作/外传钩子”（必须在正文/卷末留钩）
