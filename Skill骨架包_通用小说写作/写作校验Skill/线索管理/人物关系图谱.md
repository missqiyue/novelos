# 人物关系图谱（通用可维护模板）

> 用法：  
> - 新项目：复制本文件，先把“主干关系图”填满，再补“势力/证据线”。  
> - 长篇连载：每3章更新一次（新增关系/阶段变化/阵营迁移），避免后期“关系打架”。  

---

## 0) 图谱图例（建议统一）
- 亲缘：`父母/子女/祖孙/血缘不明`  
- 婚姻：`婚约/退婚/成婚/和离/逼婚`  
- 利益：`雇佣/供货/票号往来/盐引/漕运/军需`  
- 敌对：`构陷/追杀/灭证/反咬/掐断供给`  
- 同盟：`公开同盟/并案/互证/盟约`  
- 阶段字段（必须写）：`阶段@章节区间`（例：误会@114-160 → 并案@161-300）

---

## 1) 主干关系图（示例：锦绣回生｜卷六终局版）

> 说明：以下为**通用占位模板**（角色名/势力名请替换）。  
> 建议：用“阵营分组 + 关系带阶段/区间”的写法，保证后期可审计。

```mermaid
graph TD
  %% ===== 主家庭/亲缘 =====
  subgraph 家族_亲缘
    A[女主/主角A] -->|母女/父女/兄妹…@章-章| B[亲属B]
    A -->|外祖孙/师徒…@章-章| C[亲属/导师C]
  end

  %% ===== 感情线（先虐后甜）=====
  subgraph 情感线
    A <-->|对立@章-章<br/>合作@章-章<br/>并肩@章-章| ML[男主/关键对象]
  end

  %% ===== 反派链条（宅斗→权谋→国案）=====
  subgraph 旧网_反派
    R1[反派节点1] -->|逼婚/污名/夺产…@章-章| A
    R2[反派节点2] -->|家法/名节/宗祠…@章-章| A
    R3[外室/替罪羊/工具人] -->|污名线/继嗣文书…@章-章| R1
    G1[帮会/商会/黑网] -->|走私/断运/纵火…@章-章| A
    P1[更上层权力节点] -->|灭证/反咬/封口…@章-章| A
  end

  %% ===== 新政阵营 =====
  subgraph 新政_互证系统
    K1[权力方/官署/王府] -->|互证/立案/编号规则| SYS[制度系统/三司/审计]
    SYS -->|公开审计/处置范例| A
    A -->|资格/议席/印信/授权| SYS
    ML -->|背案/执法/托底| SYS
  end

  %% ===== 互证闭环（以纸为刃）=====
  subgraph 互证闭环_文书
    E1[证据形态1：回执/收讫链] --> A
    E2[证据形态2：号段/名单/账册] --> P1
    E3[证据形态3：医案/契书/供词] --> B
    E4[证据形态4：条款/盟约/规程] --> A
  end

  %% ===== 关键结果指向 =====
  A -->|终局结果（例：商道成国道/翻案/登基/清算）| END[新常态：编号/条款/互证成为习惯]
```
