# 情绪节拍表（每3章批次必填｜通用模板）

> 用途：解决“每章都有钩子但整体情绪平淡”的问题。  
> 原理：**情绪需要波浪起伏**——压抑→释放→新压抑，让读者在“追下去”中上瘾。  
> 使用：每3章批次写章卡前，先填本表；每章正文完成后，对照检查。

---

## 0) 情绪节拍核心规则（写死）
1) 每3章必须完成一个**【压抑→释放→新压抑】**的完整波浪。  
2) 压抑时长不超过2章（第1-2章压，第3章必须释放）。  
3) 释放不能一次给完：**70%释放+30%留白**（让读者想追下一章）。  
4) 每一批的“新压抑”必须比上一批**更高一层**（对手更强/代价更大/问题更复杂）。  

---

## 1) 情绪节拍类型（选一种用）

### A. 标准波浪型（适合日常/推进批次）
- 第1章：【压抑】对手压制 / 规则卡点 / 资源短缺 / 信任裂缝  
- 第2章：【对抗】主角反击 / 规则反用 / 代价承受  
- 第3章：【释放+新压抑】翻盘胜利（70%）+ 更大威胁浮现（30%留白）

### B. 翻转型（适合高潮批次）
- 第1章：【先扬】主角领先 / 对手弱势 / 读者以为要赢  
- 第2章：【反转】意外打击 / 队友背叛 / 规则陷阱  
- 第3章：【绝杀+新钩】绝境翻盘（代价惨重）+ 更深危机

### C. 累积型（适合情感批次）
- 第1章：【压抑】误会/裂缝/沉默  
- 第2章：【试探】想靠近又收回 / 话说一半  
- 第3章：【爆发+余震】情感决堤/和解/冲突+新问题埋下

### D. 缓升型（适合开荒/布局批次）
- 第1章：【冷淡】信息铺垫 / 局势模糊  
- 第2章：【升温】小进展 / 信心建立  
- 第3章：【突破+悬念】关键发现/里程碑+下一步未知

---

## 2) 情绪强度量化表（1-10分）

| 维度 | 定义 | 打分标准 |
|------|------|----------|
| **压抑强度** | 当前危机的严重程度 | 1=小麻烦；5=生死攸关；10=文明存亡 |
| **释放强度** | 读者获得的爽感程度 | 1=微满足；5=强烈快感；10=爆燃 |
| **新压抑预埋** | 下一批的危机等级 | 必须 > 本批释放强度的70% |

---

## 3) 每3章批次模板（直接复制填写）

```markdown

### 批次类型（选一种）
[ ] A. 标准波浪型   [ ] B. 翻转型   [ ] C. 累积型   [ ] D. 缓升型

### 节拍设计
| 章节 | 节拍节点 | 情绪动作（具体写） | 情绪强度 |
|------|----------|-------------------|----------|
| 第___章 | 【压抑/先扬/冷淡】 | ________ | ___/10 |
| 第___章 | 【对抗/反转/升温】 | ________ | ___/10 |
| 第___章 | 【释放+新压抑/绝杀/爆发】 | ________ | ___/10（+新压抑：___/10） |

### 爽点设计（每批至少1个）
- 本批爽点类型（选一）：
  [ ] 拿到东西（证据/编号/筹码/回执）
  [ ] 逼出代价（对手具名/背锅/资源损失）
  [ ] 翻转一局（规则反用/以彼之道还施彼身）
  [ ] 关系升级（并肩/信任/默认关系）
  [ ] 代价落地（伤/名声/失去/背叛）
- 本批具体爽点：________

### 新压抑设计（必须比爽点更高一层）
- 新威胁来源：________（对手变招/规则升级/外部危机/关系背叛）
- 新代价预埋：________
- 新悬念：________（让读者必须追下一批）

### 上批情绪承接
- 上批释放是否留白30%？是/否（若否，本批需补）
- 上批新压抑是否在本批开头体现？是/否（若是，写清如何体现）
```

## 5) 情绪节拍质检清单（每批写完打勾）

- [ ] 压抑→释放→新压抑的波浪完整（不缺节拍）
- [ ] 第3章/第6章/第9章有爽点（拿到/逼出/翻转）
- [ ] 释放强度70%，留白30%（不许一次给完）
- [ ] 新压抑等级 > 本批释放强度的70%（逐批升级）
- [ ] 爽点与主线/敌人链/硬点链/情感链之一绑定
- [ ] 每批至少1个具象细节（物/味/动作链/代价）
- [ ] 章末钩子与新压抑一致（不让读者猜错）
