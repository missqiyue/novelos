# 详细章卡模板

> 用途：在"章纲"和"单章正文"之间增加一个中间层，提供更详细的写作支撑。  
> 何时用：每3章或每5章批次写之前，先出详细章卡。  
> 如何用：复制本模板，按项目设定填入内容，写完单章后删除章卡中的已完成部分。

---

## 六项锚点填写说明

| 锚点 | 必须包含 |
|------|----------|
| **场景锚** | 具体地点（具体到房间/街道）+ 时间（白天/夜里/时辰） |
| **目标锚** | 一句话说明主角这章"要做什么" |
| **对抗锚** | 谁/什么在阻拦（人/规则/内心恐惧） |
| **推进点** | 这章完成后，故事必须往前推进一步 |
| **爽点** | 读者读完这章的"爽感"来源（拿到东西/打脸/翻盘/升级） |
| **钩子** | 章末的"追读理由"，必须是新问题/新威胁/新代价 |

---

## 场景分解填写说明

每个场景需要包含：

1. **动作链**：主角的行动步骤，每步一行
2. **关键对话**：需要写出来的重要对话，用引用格式
3. **细节**：感官细节（物/味/声音/触感），让场景有画面感
4. **钩子/后果**：场景结尾的悬念或代价落地

---

## 亲密戏安排填写说明

| 等级 | 定义 | 写法要求 |
|------|------|----------|
| **Lv0** | 无亲密 | 无 |
| **Lv1** | 轻微暧昧 | 眼神/语气/距离的暗示 |
| **Lv2** | 轻度亲密 | 牵手/拥抱/额头相抵 |
| **Lv3** | 中度亲密 | 吻/亲密接触但止于暗示 |
| **Lv4** | 重度亲密 | 强暗示→黑屏→后果落锤 |
| **Lv5** | 最高级 | Lv4+当章代价落地+后续影响 |

每场亲密戏必须回答：
1. 这场亲密推进了哪条故事链？
2. 这场亲密的代价是什么？
3. 第三方是否知道？（知道=冲突来源）
