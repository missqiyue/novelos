# 正文/卷01

> 本目录为正文写作层，包含单章正文文件和"详细章卡"。

---

## 写作标准流程

```
章纲（章纲_卷XX_XX.md）
    ↓
详细章卡（第XXX-XXX章详细章卡.md）
    ↓
章节内容（第XXX章_标题.md）
    ↓
写作校验（写作校验清单.md）← 强制执行
    ↓
下一章
```

> **每3章为一批次**，批次内情绪节拍遵循【压抑→释放→新压抑】波浪设计。

## 详细章卡（必做）

详细章卡是"章纲"和"单章正文"之间的中间层，提供更详细的写作支撑。

**何时用：**
- 新项目起步阶段（前20章必须用）
- 复杂剧情批次（多线并进/关键转折点）
- 写作卡顿时（详细章卡能帮你理清思路）

**参考模板：** `详细章卡模板.md`

---

## 写作校验（每章必做）

每章写完后，必须完成以下校验：

### A. 基础校验（5项必做）

| 校验项 | 检查内容 | 不通过处理 |
|--------|----------|-----------|
| 禁用破折号 | 全文搜索"——"，如有则替换 | 替换为句号"。" |
| 删除总结句 | "这一刻他明白了"等句式 | 删除 |
| 三锚开场 | 开头200字内交代【场景/目标/对抗】 | 补充 |
| 章末钩子 | 是新问题/新威胁/新代价 | 重写钩子 |
| 爽点落地 | 可验证（编号/签收/盖章） | 加验证点 |

### B. 进阶校验（选做≥2项）

- [ ] 具象细节：物/味/动作链
- [ ] 小阻碍：视线被遮/脚下打滑等
- [ ] 人物小错：按印按歪/名字写错

### C. 批次校验（每3章必做）

- [ ] 情绪节拍：【压抑→释放→新压抑】
- [ ] 主线推进：三章内推进了什么？
- [ ] 故事链：事业链/情感链/伏笔链

**详细清单：** `../写作校验Skill/写作校验清单.md`

---

## 校验不通过禁止进入下一章

```
校验通过 → 进入下一章
校验不通过 → 修改 → 重新校验 → 通过 → 进入下一章
```
